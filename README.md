# dunils-hold
Public support for Dunil's Hold. A hybrid roguelike with deterministic gameplay and on-device AI narration.

## Support & Feedback

If you encounter any bugs or have ideas for new features, please check our [Issues](https://github.com/keithduke/dunils-hold-github-support/issues) page.

- **Found a bug?** Please submit a [Bug Report](https://github.com/keithduke/dunils-hold-github-support/issues/new?template=bug_report.md).
- **Have a feature idea?** Please submit a [Feature Request](https://github.com/keithduke/dunils-hold-github-support/issues/new?template=feature_request.md).
